# HMchat项目笔记

##项目描述

* 冯慧敏的第一个项目

* 使用Vue2进行的仿手机QQ的webapp的制作

* 项目地址 `https://github.com/jiangqizheng/vue-MiniQQ`

## 项目已实现功能

* 对话功能——想着既然是QQ总要能进行对话交流，所以在项目中接入了图灵聊天机器人，可以与列表中的每个人物进行对话。

* 左滑删除——左滑删除相关消息。

* 搜索页面——点击右上角搜索按钮，能够进入搜索页面，输入对应的单词或者数字，动态查找好友。

* 项目中数据流动由vuex进行控制
